## 三种模式，覆盖不同使用场景

Manico 支持三种工作模式，不同类型的用户各取所需：

**Dock 模式**：自动映射你的程序坞。你 Dock 里有什么，Manico 启动条里就显示什么，按位置对应数字键直接切。适合刚上手、不想做太多配置的用户。

**切换器模式**：映射当前正在运行的 App，相当于增强版的 `Command + Tab`。适合同时开很多软件的重度多任务用户。

**自定义模式**：完全由你来排布每个快捷键对应哪个 App，还可以为不同 App 指定不同的激活键。这是键盘党的最爱，配置好之后，开什么软件都只需要两个按键。
